# Personal website — Xinyue (Cindy) Wang

[![Deploy to GitHub Pages](https://github.com/newmoooon/personalwebsite/actions/workflows/deploy-pages.yml/badge.svg)](https://github.com/newmoooon/personalwebsite/actions/workflows/deploy-pages.yml)

Personal portfolio site — React + Vite.

**Live: <https://newmoooon.github.io/personalwebsite/>**

## Develop

```bash
cd cindywang
npm install
npm run dev      # local dev server
npm run build    # production build → cindywang/dist
```

## Deploy

Push to `main` — GitHub Actions builds the site and publishes it to GitHub
Pages automatically. No manual steps, no committed build artifacts.
